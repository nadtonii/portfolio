Stack

The best possible way to manage your tasks.

Role

Designer, Developer

Timeline

2 months

Tools

Figma, Xcode, Notion

Overview

Stack is a task management app that allows you to create, edit, and delete tasks. It also allows you to create tasks with natural language processing.

Stack allows you to create groups, set reminders, track deadlines, and more.

The challenge

The main challenge was to create a task management app that is easy to use and navigate. The app also needed to be able to handle a large number of tasks and reminders. Personal challenge was to learn Swift and Xcode and transfer the design from Figma to Swift.

Personal challenge was to learn Swift and Xcode and transfer the design from Figma to Swift.

The process

Research

Conducted user research through surveys and competitor analysis. Studied popular task management apps like Todoist, Things, and TickTick to identify key features and opportunities for improvement.

Design

Created high-fidelity prototypes in Figma, focusing on intuitive task creation and management. Conducted multiple rounds of usability testing to refine the interface and user flows.

Development

Built the app using Swift and Xcode, implementing core features like natural language processing and reminders. Refined the UI components to match the Figma designs perfectly.

The solution

Brining a natural typing experience to the users. Creating an intuitive experience for the users to create tasks and manage them. Making the app more user friendly and easier to use than competitors.
