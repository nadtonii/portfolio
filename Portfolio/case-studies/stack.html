Stack

The best possible way to manage your tasks.

Role

Designer, Developer

Timeline

2 months

Tools

Figma, Xcode, Notion

Overview

Stack is a task management app that allows you to create, edit, and delete tasks. It also allows you to create tasks with natural language processing.

Stack allows you to create groups, set reminders, track deadlines, and more.

The challenge

The main challenge was to create a task management app that is easy to use and navigate. The app also needed to be able to handle a large number of tasks and reminders.

Personal challenge was to learn Swift and Xcode and transfer the design from Figma to Swift.

The process

Research

Through user research, I identified three main problems with existing task management apps:

  • Users found it frustrating to input tasks with multiple steps and details
  • Most apps required too many clicks to create a simple task
  • Important tasks often got lost in long lists

I solved these by implementing natural language processing for quick task creation, smart categorization, and an intuitive priority system.

Design

The design process focused on solving key user pain points:

  • Created a natural language input that understands dates, priorities, and categories
  • Designed smart lists that automatically organize tasks based on context
  • Developed a minimal interface that keeps the focus on task management

Multiple rounds of user testing helped refine these features until users could create and manage tasks with minimal friction.

Development

Development presented several technical challenges:

  • Built a custom natural language parser for task creation
  • Implemented local data storage for offline functionality
  • Created a notification system for reminders and deadlines
  • Optimized performance for smooth task management experience

The app was built using Swift and SwiftUI, ensuring native performance and iOS design guidelines compliance.

The solution

Brining a natural typing experience to the users. Creating an intuitive experience for the users to create tasks and manage them. Making the app more user friendly and easier to use than competitors.
